Project Description :

This project was created in January 2026 and incorporates a number of different techniques, mainly:

  • Landing modal
  • Real-time clock (top left of the site)
    • With the associated Javascript
  • Switches to enable or disable various sections of the site
    • Also with the associated Javascript
  • Fixed left hand-side menu
    • With quick links to :
      • Navigate to site TOP
      • Send me an email
      • Send me a Whatsapp message
      • Navigate to site END
  • Animated <a> tags in <nav>
  • <main> set to GRID display with :
    • 30% set for <aside>
    • auto set for #mainContainer
  • with <aside> loading a thumbnail which expands in #mainContainer
    • And associated Javascript
  • Section with spanned images
    • That expand on hover
    • CSS linear gradient on the bottom to fade the images
    • Clicking expanded image loads the image in a new tab

  • Please feel free to view the page source for the HTML, CSS and Javascript codes I used
  • Appreciated if you follow my Github page in return to using my code
